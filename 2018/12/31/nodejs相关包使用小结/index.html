nodejs相关包使用小结

Dec 31, 2018nodejs

此文主要是为了举例 nodejs 的一些相关包用法总结，没有 nodejs 环境可以先到官网下载安装。nodejs 实质就是写 js 文件，根据一些包快速开发。这里只是列举其中一小部用法，详细的还需要各位去看看官网的 API，这里只是提供给大家做个了解。

1、http 模块

2、fs 模块

3、querystring 模块

4、url 模块

5、express 模块

6、body-parser 模块

  • 中间件，用于处理 JSON, Raw, Text 和 URL 编码的数据

7、cookie-parser 和 cookie-session 模块

  • 解析 Cookie 的工具，通过 req.cookies 可以取到传过来的 cookie，并把它们转成对象。

8、ejs 和 jade 模板引擎

9、multer 和 path 模块

  • 作为中间件进行数据处理，例如对附件上传下载。

10、consolidate 模块

  • consolidate 模块可以说是一个各种模板引擎的结合体。它包括了我们之前的 jade 和 ejs。通过配置就可以使用多种模板引擎。

11、mysql 模块

12、package 管理
