Vue1.0学习小结2

Aug 05, 2018vue

vue 是一个 mvvm 框架(库)，它和 angular 类似，但相对比较容易上手、小巧。具体详情可以查阅vue 官网

1.生命周期

  • 用 Vue 框架，熟悉它的生命周期可以让开发更好的进行。这里借助一个图片说明一下生命周期的钩子函数。

2.计算属性

3.自定义方法和属性

4.数据监听

  • 可以在数据发生变化的时候监测处理，类似 angular1 的脏处理
  • a、vm.$watch(name,fnCb); //浅度监听针对基本值类型
  • b、vm.$watch(name,fnCb,{deep:true}); //深度监视，可以处理 json 对象

5.动画

  • vue 动画有格式参考，例如想加 fade 动画效果，可以有 fade-transition/fade-enter/fade-leave 等，不过如果想弄点好看点，建议配合 animate.css

6.组件

  • 使用组件的方式有多种，还可以嵌套（父子级）。如果想了解组件之间的关系，可以安装 vue-devtools 插件，它可以从 chrome 商店直接下载安装。不过要注意的一点就是，需要翻墙才能下载。

7.slot

  • 它的作用是作用: 占个位置，类似 ng 里面 transclude（指令），可以参考插件vue-infinite-loading

8. 路由

  • 对于单页应用，官方提供了 vue-router 进行路由跳转的处理
